# 🎓 VideoBelajar

VideoBelajar adalah platform pembelajaran video berbasis web yang memungkinkan pengguna mengakses berbagai kursus dan materi pembelajaran secara online. Aplikasi ini dibangun menggunakan **React JS** dengan desain modern, responsif, dan ramah pengguna.

## 📖 Tentang Project

VideoBelajar dirancang untuk membantu pengguna menemukan dan mempelajari berbagai topik melalui video pembelajaran berkualitas. Platform ini menyediakan daftar kursus unggulan lengkap dengan informasi instruktur, rating, dan harga kursus.

## 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/Panji-Kusumah/Video_Belajar.git
```

Masuk ke folder project:

```bash
cd videobelajar
```

Install dependencies:

```bash
npm install
```

Jalankan development server:

```bash
npm run dev
```

Buka browser:

```bash
http://localhost:5173
```
